# AI Chatbot Resume Analyzer

An AI-powered Resume Analyzer built using Python and Streamlit.  
This project helps users upload their resume and get smart analysis, suggestions, and feedback.

## Features

✅ Upload Resume (PDF)  
✅ Resume Text Extraction  
✅ Skills Analysis  
✅ Suggestions for Improvement  
✅ Simple & Clean UI  
✅ AI Based Resume Review  

## Technologies Used

- Python
- Streamlit
- PyPDF2
- NLP
- Machine Learning

## How to Run

```bash
pip install streamlit PyPDF2
streamlit run app.py
